
## 1. Roo Code (구 Roo Cline) 시리즈
질문하신 **Codemate**는 아마 **Roo Code**의 특정 모드나 유사한 에이전트 도구를 말씀하시는 것 같습니다. 여기서 핵심은 **Roo Code(전 Cline)**입니다.
 * **Roo Code (구 Cline):** VS Code 내부에 설치하는 **'에이전트'** 확장 프로그램입니다. 단순히 코드를 짜주는 게 아니라, "로그인 기능 만들어줘"라고 하면 **직접 파일을 만들고, 터미널 명령어를 치고, 에러가 나면 스스로 고치는** 등 사람처럼 행동합니다.
 * **Codemate / Roo 관련 설정:** Roo Code 안에서 특정 AI 모델(Claude, GPT 등)을 연결해 '친구'처럼 부리는 설정을 커스터마이징한 형태라고 이해하시면 됩니다.
## 2. Cursor (커서) - AI 전용 IDE
 * **정체:** VS Code를 통째로 복제해서 만든 **AI 전용 코드 에디터**입니다.
 * **특징:** 확장 프로그램으로 설치하는 게 아니라, 프로그램 자체가 AI를 위해 설계되었습니다. Ctrl + K(코드 생성), Ctrl + L(채팅) 기능이 매우 강력하고 빠릅니다. 현재 개발자들 사이에서 **가장 만족도가 높은 도구** 중 하나입니다.
## 3. Claude Code & Claude (Anthropic)
 * **Claude Code:** Anthropic사에서 만든 **터미널(CLI) 전용 AI 도구**입니다. VS Code 화면이 아니라 검은색 터미널 창에 명령어를 입력해서 코드를 수정합니다. 전문가용 도구에 가깝습니다.
 * **Claude 3.5 Sonnet:** 위 모든 도구(Roo, Cursor 등)의 **'두뇌'** 역할을 하는 가장 똑똑한 AI 모델 이름입니다. 코딩 실력이 워낙 뛰어나서 대부분의 도구가 이 모델을 사용합니다.
## 4. 한눈에 비교하는 도구 성격

| 구분 | 도구 이름 | 특징 | 추천 대상 |
| :--- | :--- | :--- | :--- |
| **에이전트** | **Roo Code (Cline)** | 내가 시키는 대로 파일을 직접 만들고 실행함 | VS Code를 그대로 쓰며 강력한 대행자를 원하는 분 |
| **에디터** | **Cursor** | AI와 에디터가 물아일체된 최고의 속도와 편의성 | 가장 쾌적한 AI 코딩 환경을 새로 구축하고 싶은 분 |
| **터미널형** | **Claude Code** | 채팅창 없이 명령어로만 코딩 작업 수행 | 터미널 환경이 익숙한 숙련된 개발자 |

### 5. LLM 모델의 종류: Claude 시리즈
- **Opus (오퍼스):** Claude 3 시리즈 중 가장 지능이 높은 최상위 모델. 복잡한 추론과 설계에 강점.
- **Sonnet (소네트):** 코딩 생산성 도구에서 가장 많이 쓰이는 표준 모델. (3.5 버전이 특히 유명)
- **Haiku (하이쿠):** 가볍고 빠른 처리를 위한 모델.
